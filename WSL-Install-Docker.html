此教程只适用于Windows 10 非 Home 版本

配置 Debian(WSL)

打开 Debian 并初始化设置完毕后，推荐更换Apt-get国内镜像源；

  1. 删除旧版本Docker，使用以下命令：

    xxxxxxxxxx
    $ sudo apt-get remove docker \
                   docker-engine \
                   docker.io

  2. 使用apt，安装HTTPS传输软件包与CA证书

    xxxxxxxxxx
    $ sudo apt-get update
    $ sudo apt-get install \
         apt-transport-https \
         ca-certificates \
         curl \
         gnupg2 \
         lsb-release \
         software-properties-common

  3. 安装软件源的 GPG key

    鉴于国内网络问题，推荐使用国内镜像源，官方源请在注释中查看

    xxxxxxxxxx
    $ curl -fsSL https://mirrors.ustc.edu.cn/docker-ce/linux/debian/gpg | sudo apt-key add -
    ​
    # 官方源
    # $ curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -

  4. 添加软件源并更新

    此为稳定版本的命令，如若需要测试版或每日构建版，请将"stable"更改为"test"或"nightly"

    xxxxxxxxxx
    $ sudo add-apt-repository \
       "deb [arch=amd64] https://mirrors.ustc.edu.cn/docker-ce/linux/debian \
       $(lsb_release -cs) \
       stable"
    $ sudo apt-get update
    ​
    # 官方源
    # $ sudo add-apt-repository \
    #    "deb [arch=amd64] https://download.docker.com/linux/debian \
    #    $(lsb_release -cs) \
    #    stable"
    # $ sudo apt-get update

  5. 使用自动安装脚本

    来自官方的便携脚本，会将安装 Docker-CE 的一切事项与准备搞掂

    xxxxxxxxxx
    $ curl -fsSL get.docker.com -o get-docker.sh
    $ sudo sh get-docker.sh --mirror Aliyun

  6. 将自己所在用户添加进组中：

    xxxxxxxxxx
    $ sudo usermod -aG docker $USER
    # $USER 应当改为你当前在 Debian 所用的用户名

  7. 更改Docker CE Server端默认连接端口

    用 Vim 或 Nano 打开 ~/.bashrc

    将以下代码在末尾处添进：

    xxxxxxxxxx
    export DOCKER_HOST=tcp://0.0.0.0:2375

    保存并刷新：

    xxxxxxxxxx
    $ source ~/.bashrc
